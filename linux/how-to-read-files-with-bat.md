
# How to read files with bat

## What is bat
可以打印并且合并文件的命令。

cat 的复制品，外加语法高亮和 Git 集成。

更多信息：https://github.com/sharkdp/bat.

## How do I use it?

1. Install Command
```shell
# on mac
> brew install bat
```

2. Run
```shell
> bat test.md
```
